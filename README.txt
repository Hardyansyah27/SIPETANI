Cara pakai:
1. Ekstrak folder ini.
2. Buka file index.html di browser (klik dua kali).
3. Menu di kiri akan memuat halaman pada area kanan.
4. Untuk mengaktifkan Chatbase chatbot di halaman asisten.html:
   - Buka file asisten.html dan cari komentar yang berisi contoh script embed.
   - Tempelkan script Chatbase (Deploy -> Chat Widget -> Installation) ke asisten.html
     atau cukup isi file custom-chatbase.js dengan script embed (saran).
5. Untuk mengganti logo, ganti file logo.svg dengan gambarmu (jangan ubah nama file)
6. Untuk mengganti nomor WhatsApp, buka kontak.html atau gunakan tombol 'Kontak' di header.
Catatan: Detector menggunakan heuristik warna sederhana untuk demo.
